Beginner step-by-step guide (exact commands)

1) Prerequisites (install if you don't have)
   - Node.js (LTS). Check: node -v && npm -v
   - MongoDB (installed and running locally). Check: mongo --eval "db.runCommand({ ping: 1 })"
   - VS Code (recommended) or any editor.

2) Unzip the archive and open a terminal.
   Example (Windows): Right-click -> Extract All...
   Example (macOS/Linux): double-click or use `unzip auth-module.zip`

3) Start Backend
   cd auth-module/backend
   copy .env.example to .env (Windows: copy .env.example .env ; macOS/Linux: cp .env.example .env)
   Edit .env and set JWT_SECRET (a simple secret is fine for local testing)
     PORT=5000
     MONGO_URI=mongodb://localhost:27017/auth_db
     JWT_SECRET=your_secret_here
   Install deps:
     npm install
   Start server:
     npm start
   You should see "Server started on port 5000" and "MongoDB connected"

4) Start Frontend (in a new terminal)
   cd auth-module/frontend
   copy .env.example to .env
     REACT_APP_API_URL=http://localhost:5000
   Install deps:
     npm install
   Start app:
     npm start
   React will open at http://localhost:3000

5) Use the App
   - Register: http://localhost:3000/register
   - Login: http://localhost:3000/login
   - Profile: http://localhost:3000/profile (protected)

6) Troubleshooting
   - MongoDB connection: make sure mongod is running (or run `sudo systemctl start mongod` on Linux)
   - Port in use: change PORT in backend/.env
   - Frontend not calling backend: ensure REACT_APP_API_URL matches backend address
